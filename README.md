# IdleGame

클릭형 아이들 게임 (Clicker Idle Game)

## 기술 스택

- **Engine**: Unity 2D (URP)
- **Language**: C#
- **Version Control**: Git / GitHub

---

## 게임 개요

화면의 몬스터를 클릭해 데미지를 주고, 처치 시 골드·아이템을 획득한다.
골드로 스킬을 업그레이드하고 아이템을 장착해 스탯을 강화한다.
스테이지가 올라갈수록 몬스터가 강해지고 보스가 등장한다.

**코어 루프:** 클릭 → 처치 → 보상 → 강화 → 반복

## 현재 구현 상태

### 핵심 시스템

| 시스템 | 상태 | 설명 |
|--------|------|------|
| 몬스터 | ✅ | HP, 데미지, 드랍, 파티클, 클릭 처리 |
| 스테이지 | ✅ | 스테이지 진행, 보스 등장, 스테이지 선택 팝업 |
| 도망 시스템 | ✅ | 골드 소모로 현재 몬스터 스킵 |
| 플레이어 스탯 | ✅ | 클릭/자동 데미지, 공속, 골드배율, 드랍률 |
| 재화 | ✅ | 골드, 보석 관리 |
| 업그레이드 | ✅ | 스킬 업그레이드 구매·효과 적용 |
| 인벤토리 | ✅ | 아이템 보유·장착·판매·드랍 |
| 파티클 | ✅ | JSON 기반 이펙트 스폰 |
| 오디오 | ✅ | BGM·SFX 재생 |
| 설정 | ✅ | 볼륨·진동·알림·데이터 초기화 |
| 저장 | ✅ | PlayerPrefs 기반 자동 저장 |

### UI

| 패널 | 상태 | 설명 |
|------|------|------|
| 전투 HUD | ✅ | 체력바, 몬스터 이름, 스테이지 표시 |
| 업그레이드 패널 | ✅ | 스킬 목록 + 3열 보너스 요약 |
| 장비 패널 | ✅ | 장비 목록 + 3열 최종 스탯 요약 |
| 상점 패널 | ✅ | 구매/판매 탭 |
| 설정 팝업 | ✅ | 볼륨·토글·초기화·개발자 패널 |
| 아이템 토스트 | ✅ | 획득 시 등급별 색상 알림 |
| 스테이지 선택 | ✅ | 해금된 스테이지 선택 팝업 |
| 업적 패널 | 🚧 | 준비중 |

## 프로젝트 구조

```
Assets/
├── Scripts/
│   ├── Core/           # Monster, UIBoot, BackgroundManager
│   ├── Data/           # ScriptableObject 데이터 정의
│   ├── Managers/       # 도메인 매니저 (PlayerStats, MonsterManager 등)
│   ├── UI/
│   │   ├── UITheme.cs  # 색상/스타일 중앙 관리 ★
│   │   ├── UIHelper.cs # UI 생성 유틸
│   │   ├── Panels/     # 각 탭 패널 UI
│   │   └── Elements/   # 체력바, 토스트 등 개별 요소
│   ├── Gameplay/       # TapController
│   └── Utils/          # NumberFormatter
├── Scenes/
│   └── Main.unity
├── ScriptableObjects/
│   ├── Upgrades/
│   └── Items/
├── Sprites/
├── Audio/
└── Resources/
    ├── Backgrounds/    # 스테이지별 배경
    └── Particles/      # 파티클 이펙트 JSON
```
